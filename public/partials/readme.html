Motion Rest Server

Pre-requisites

Motion should be configured to write to a MySQL or PostgreSQL
Node.js and NPM
Bower

Introduction

This program serves the motion detection events captured by Motion.

Installation

To install motion, execute the following commands:

  • git clone https://github.com/scrussell24/motion-rest-server.git
  • cd motion-rest-server
  • npm install
  • bower install

Edit the config/config.json with your db info.

To run the server execute bin/www.The server should be running on port 3000.
